# Screening Note 生成器

## 總覽

根據「Screening Note（通用模板 V1，跨公司適用）.docx」模板，針對輸入的公司名稱，自動搜集公開資料並生成一份結構化的 Screening Note Word 文件。

## 模板位置

同目錄下的 `Screening Note（通用模板 V1，跨公司適用）.docx`

## 使用方式

1. 讀取本目錄下的 docx 模板，了解報告結構與欄位
2. 針對目標公司，透過 web search 搜集以下資料：
   - **公司治理面**：董事會組成、獨立董事比例、審計委員會、薪酬委員會、公司治理評鑑成績
   - **股權結構面**：大股東結構、董監持股、質押比例、外資持股、關聯交易
   - **營運面**：主要業務、營收結構、產業地位、競爭優勢、近期重大事件
   - **財務面**：近三年營收與獲利趨勢、負債比率、現金流狀況、股利政策
3. 依照模板格式，生成 Word 文件（.docx）
4. 每筆資料標註來源出處

## 輸出格式

- 檔案格式：.docx（Word 文件）
- 命名規則：`{公司名稱}_Screening_Note.docx`

## 注意事項

- 所有數據必須附上來源（公開資訊觀測站、年報、新聞等）
- 財務數據以最近一個完整會計年度為主
- 治理評鑑以最新公布結果為準
- 如資料不可得，標註「N/A — 待補充」
