# Deep Research Prompt Template

以下 prompt 用於資料搜集。將 `{TARGET_NAME}` 替換為目標人物姓名，`{COMPANY_NAME}` 替換為其主要公司名稱，`{STOCK_CODE}` 替換為股票代號。

---

## Prompt（複製貼上即可）

```
請針對【{TARGET_NAME}】（{COMPANY_NAME}，股票代號：{STOCK_CODE}）進行全面的人脈網絡調查，使用 web_search 從維基百科、天下雜誌、今周刊、鏡週刊、工商時報、經濟日報、Yahoo新聞、公開資訊觀測站等來源交叉驗證。若無法找到某項資訊，請註明「N/A」。

請按照以下 8 大類別整理資料，最終輸出為 JSON 格式：

---

### 1. 個人介紹
- 出生年份與出生地
- 完整學歷（學校、科系、學位、年份）
- 職涯關鍵里程碑（15-25個，含年份與事件描述）
- 現任所有職務
- 持股資訊（股數、比例、質押率）
- 投資興趣/專注領域
- 經營哲學/經典語錄（3-5句，含出處）
- 個人風格特質描述

### 2. 朋友圈（8 類，每類 3-8 人）

每人需包含：姓名、現職、關係判斷（🟢正向/🟡中性/🔴負向/⚪不明）、關鍵互動脈絡（20-40字）、資料來源URL。

**2.1 核心圈（個人）**：配偶、摯友、創業夥伴、交惡對象
**2.2 企業圈**：董事會成員、經營團隊、重要幹部
**2.3 協會與政商圈**：公協會角色、基金會、政界人脈
**2.4 政策與產業推手圈**：產業領袖、競爭對手、供應鏈
**2.5 學術校友圈**：母校連結、校友網絡
**2.6 思想與論壇引路人圈**：思想影響者、論壇互動
**2.7 媒體與公開平台圈**：重要媒體事件（正面+負面，含時間）
**2.8 迴避對象與引薦建議**：
  - 迴避名單（🔴，含原因）
  - 引薦管道（🟢，含建議方式）

### 3. 家族關係人（四親等內）

按世代整理：
- G0（祖父母輩）
- G1（父母與手足輩）
- G2（本人、配偶、兄弟姊妹）
- G3（子女與子女配偶）

每人需包含：姓名、與主角關係、出生年份、學歷、現職、職涯經歷、來源。
找不到的標「N/A」，但仍需列出（如「配偶：姓名N/A」）。

### 4. 家族企業版圖
- 家族成員在企業中的任職一覽（成員、關係、職務、參與程度）
- 家族關聯投資控股公司（公司名、代表人、說明）
- 各公司持股比例

### 5. 媒體事件時間軸
- 正面事件（時間、媒體/平台、內容摘要）
- 負面事件（時間、媒體/平台、內容摘要）

### 6. 總結分析
- 正面觀察（5點）
- 風險關注（5點）
- 建議追蹤（5點）
- 關鍵結論（一段約100字的綜合評述）

---

**輸出格式**：請直接輸出結構化的 JSON，格式參照 data_schema.json。
**資料準確性**：每筆資料必須附上來源。優先使用：公開資訊觀測站 > 年報 > 財經媒體 > 維基百科。
```
