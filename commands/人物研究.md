你現在要針對 **$ARGUMENTS** 生成一份完整的人脈網絡報告 PPT。

請觸發 `network-report-skill`，依照該 skill 的完整工作流程執行：

1. **Deep Research（Step 1）**：使用 Claude in Chrome 的瀏覽器控制，依序在 Gemini 和 Perplexity 上貼入 research prompt 進行深度研究，再用自己的 web search 補充搜集。目標人物為 **$ARGUMENTS**。

2. **整合研究結果（Step 2）**：將三個來源（Gemini + Perplexity + web search）的結果合併交叉驗證，搜集 8 大類資料：
   - 個人介紹（學歷、職涯 timeline、經營哲學語錄、持股）
   - 朋友圈 8 類人脈（核心圈、企業圈、協會與政商圈、政策與產業推手圈、學術校友圈、思想與論壇引路人圈、媒體與公開平台圈、迴避對象與引薦建議）
   - 家族關係人（G0-G3 四親等內）
   - 家族企業版圖
   - 媒體事件（正面/負面）
   - 家族樹結構（tree_data）
   - 總結分析

3. **整理成 JSON（Step 3）**：按照 `references/data_schema.json` 格式整理，確保來源欄位用結構化格式（title + url）、family.tree_data 完整填入、overview_cards 8 張全填。

4. **生成 PPT（Step 4）**：執行 `python scripts/generate_network_report.py` 生成 18 頁完整報告。

輸出檔案命名：`$ARGUMENTS_人脈網絡報告.pptx`
