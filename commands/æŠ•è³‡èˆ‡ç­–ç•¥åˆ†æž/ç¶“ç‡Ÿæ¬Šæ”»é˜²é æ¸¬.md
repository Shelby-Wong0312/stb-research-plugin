## Part 1：攻擊者側寫與模式分析

---

#### 【1.1 攻擊者基本資料】

- 攻擊者的全名、背後的實際控制人 / 家族
- 攻擊者的核心企業版圖：旗下有哪些公司？主要業務為何？
- 攻擊者的資金來源與財務實力估計
- 攻擊者在台灣資本市場的整體持股版圖（已知的持股清單）

#### 【1.2 歷史攻擊案例分析】

請逐一列出攻擊者過往所有已知的經營權攻擊案例，每一個案例分析以下項目：

- **標的企業**：被攻擊的企業名稱、產業別、當時的市值
- **攻擊時間軸**：從開始布局到最終結果的完整時間序列
- **攻擊手法**：
  - 如何買進股票？（公開市場收購、大量申報、透過人頭戶、透過關聯公司等）
  - 持股比例的爬升過程
  - 是否有發動委託書收購？
  - 是否有提出董事候選人？
  - 是否有公開喊話或媒體戰？
  - 是否有聯合其他股東或組成聯盟？
- **攻擊目的**：
  - 取得經營權（董事長 / CEO 位置）？
  - 取得董事席次以影響決策？
  - 逼迫公司改變策略（例：提高股利、出售資產、下市私有化）？
  - 短期炒作股價獲利了結？
- **標的企業的防禦手段**：
  - 是否有引進白騎士（White Knight）？
  - 是否有實施庫藏股回購？
  - 是否有修改公司章程（例：提高董事改選門檻）？
  - 是否有策略聯盟或交叉持股安排？
  - 是否有訴諸法律手段？
- **最終結果**：攻擊成功 / 失敗 / 和解 / 進行中？
- **案例啟示**：這個案例揭示了攻擊者的什麼偏好或模式？

#### 【1.3 攻擊模式歸納（Pattern Analysis）】

綜合所有歷史案例，歸納攻擊者的行為模式：

- **選擇標的的偏好**：
  - 偏好什麼產業？（傳產、金融、科技、營建等）
  - 偏好什麼規模的企業？（市值範圍）
  - 偏好什麼樣的股權結構？（大股東持股偏低、股權分散等）
  - 偏好什麼樣的經營狀況？（績優股、轉機股、資產股、低估值股等）

- **攻擊時機偏好**：
  - 在什麼市場環境下出手？（牛市 vs 熊市）
  - 通常在股東會前多久開始布局？
  - 是否有季節性的規律？

- **攻擊手法偏好**：
  - 慣用的持股建立方式（慢慢買 vs 大量突擊）
  - 慣用的攻擊路線（委託書戰 vs 公開收購 vs 私下協商）
  - 是否慣用法律戰或媒體戰？
  - 是否有固定的「盟友」或合作夥伴？

- **攻擊後的行為**：
  - 取得經營權後，通常做什麼？（更換管理層、出售資產、提高股利、策略轉型等）
  - 若攻擊失敗，通常如何退場？（出售持股、和解換取董事席次、繼續等待機會）

#### 【1.4 攻擊者的弱點與限制】

- 攻擊者的資金能力上限估計
- 攻擊者過去失敗的案例中，敗因是什麼？
- 監管機構對攻擊者的態度或限制
- 攻擊者的公眾形象與聲譽是否構成障礙？

## Part 2：潛在目標掃描與預測

---

#### 【2.1 篩選模型建立】

根據 Part 1 歸納出的攻擊模式，建立一個「被攻擊風險評分模型」，篩選條件包括但不限於：

- **股權分散度**：大股東持股比例低於 X%（根據歷史案例設定門檻）
- **市值區間**：落在攻擊者偏好的市值範圍內
- **產業別**：落在攻擊者偏好的產業範圍內
- **經營狀況**：
  - 股價長期低於淨值（P/B < 1）
  - 手上有大量閒置資產（土地、不動產、轉投資）
  - 配息率長期偏低（帳上現金多但不配息）
  - 近年經營績效下滑
- **治理漏洞**：
  - 董事會缺乏獨立性
  - 長期未改選董事
  - 缺乏有效的反併購防禦機制
- **其他線索**：
  - 攻擊者或其關聯人是否已持有該公司少量股份？
  - 近期是否有異常的股價波動或成交量放大？
  - 是否有內部經營權交接或家族紛爭的跡象？

#### 【2.2 風險評分與排名】

- 對台灣上市櫃企業進行掃描，列出 **風險最高的 Top 20 企業**
- 每家企業列出以下資訊：

| 欄位 | 說明 |
|------|------|
| 排名 | 風險排名 |
| 公司名稱 | 股票代碼 |
| 產業別 | — |
| 市值 | 億元 |
| 大股東持股比例 | % |
| P/B | 股價淨值比 |
| 近 3 年配息率 | 平均 % |
| 閒置資產 | 是否有大量土地/不動產/轉投資 |
| 治理弱點 | 簡述 |
| 已知持股線索 | 攻擊者或關聯人是否已持有 |
| 風險評分 | 滿分 100 分 |
| 風險等級 | 🔴 極高 / 🟠 高 / 🟡 中 |

#### 【2.3 Top 5 深度分析】

針對風險最高的 Top 5，每一家進行深度分析：

- **為什麼這家公司可能被盯上？**
  - 股權結構的具體弱點
  - 資產面的吸引力（土地、轉投資的隱含價值）
  - 經營面的問題（績效不佳、管理層爭議）
  
- **攻擊者最可能的攻擊路線？**
  - 預測持股建立方式
  - 預測攻擊時機（下一次股東會？董事改選年？）
  - 預測攻擊手法（委託書戰 vs 公開收購）

- **該企業目前的防禦能力評估**
  - 大股東的團結程度
  - 是否有反併購機制（黃金降落傘、毒藥丸條款在台灣法規下的可行性）
  - 管理團隊的應變能力
  - 是否有潛在的白騎士人選？

- **白騎士防禦策略建議**
  - 如果我方要擔任白騎士，介入的最佳時機點是什麼？
  - 建議的持股建立方式（公開市場收購、私募、策略聯盟）
  - 與現任經營團隊合作的可能架構
  - 預估需要投入的資金規模
  - 白騎士介入後的價值創造機會

### 四、報告格式要求

1. **語言**：繁體中文
2. **結構**：嚴格按照上述順序撰寫（Part 1 → Part 2 → 綜合分析）
3. **每個子項目**：
   - 先列出「關鍵發現」（Key Findings），用 1-2 句話總結
   - 再展開「詳細分析」
4. **資料來源**：標註資料來源（公開資訊觀測站、大量持股申報資料、法院判決書、新聞報導等）
5. **資訊不足時**：標註「⚠️ 資料不足，建議進一步查證」
6. **風險評分**：所有評分需說明計算邏輯
7. **台灣特有資料來源建議優先使用**：
   - 公開資訊觀測站（MOPS）— 大量持股申報、董監持股變動
   - 台灣證券交易所 — 融資融券餘額、外資買賣超
   - 經濟部商業司 — 公司登記、董監事名單
   - 司法院 — 相關訴訟紀錄
   - 集保結算所 — 股權分散表

### 五、特別注意事項

- 經營權攻防涉及高度敏感資訊，請嚴格基於公開資訊進行分析，不要編造
- 台灣的反併購法規環境與歐美不同（例如：台灣沒有毒藥丸條款），分析時需考慮台灣法規的特殊性
- 攻擊者常透過人頭戶或關聯公司持股，公開資訊可能無法完整反映實際持股，請在分析中說明此限制
- 委託書規則（台灣證券交易法）對攻防雙方的限制需納入分析
- 所有預測都需標註信心度與假設條件
